> 简介
=================================

  SAO Utils 是一款拥有 SAO（刀剑神域）外观风格的启动器，搭载各种各样强大的小工具。
  它所设计的插件模式，可以让任何的开发者为其设计插件并让其自动加载。
  这就是它以 "SAO Utils" 命名的原因了 XD

  项目主页: http://sao.gpbeta.com/
  开发日志: http://www.gpbeta.com/post/develop/sao-utils/
  Bug 反馈: http://trac.gpbeta.com/


> 支持平台
=================================

  Microsoft Windows 2000/XP/Vista/7/8/8.1/10 (32bit/64bit)


> 使用简介
=================================

  1. 运行 "SAO Utils.exe"

  2. 呼出启动器(菜单)的方法:
    a.鼠标手势: 按住鼠标 左键->右键->向下滑动一定距离.
    b.自定义热键.
    c.单击任务栏图标.
    d.重复运行 "SAO Utils.exe"

  3. 添加自定程序到启动器(菜单):
    a.点击启动器左上菜单图标
    b.取消"置顶显示"选项
    c.从文件管理器拖拽文件到启动器菜单即可.

  4. 编辑启动器(菜单):
    a.长按需要编辑的图标(非锁定模式), 带有 "Modify" 标签的按钮弹出.
    b.点击该按钮则编辑当前项目, 再长按该按钮则弹出添加/排序/删除等操作按钮.
    c.绑定虚拟目录或真实目录的项目菜单都可以进行编辑.

  5. 插件平台:
    SAO Utils 当前版本拥有15个插件:
    a.HP Bar: 以 HP 计量表显示各种数据.
    b.Image Widget: 在桌面上显示自定义的静态或动态图片. 
    c.Quick launch: 在桌面上显示一个便捷功能按钮.
    d.YUI-MHCP001: 基于 Cardinal 的桌面辅助精灵. (计划)
    e.Bangumi: 可查询每日新番放送列表的桌面挂件.
    f.Mail Box: 包含在线用户列表的邮件收发及聊天客户端.
    g.GGO Widget: 可以自定义各种数据的桌面挂件.
    h.GGO Theme: 可选的 Gun Gale Online 启动器风格.
    i.Weather Data Extension: 可在 HP-Bar 或 GGO Widget 中使用的天气数据.
    j.Sensor Data Extension: 可在 HP-Bar 或 GGO Widget 中使用的传感器数据.
    k.Web Browser Engine: 提供内置标签式浏览器支持.
    l.Web Widget: 在桌面上显示自定义的网页.
    m.Music Player Support: 提供各种常见播放器的信息及控制支持. (by RangerCD)
    n.Pdh Data Extension: 提供系统'性能计数器'中的数据. (by RangerCD)
    o.Shortcut Keys Extension: 允许用户定义一些组合键. (by RangerCD)
    插件安装:
    首选项-插件-安装... 然后选择 .NVG 后缀的插件包即可.
